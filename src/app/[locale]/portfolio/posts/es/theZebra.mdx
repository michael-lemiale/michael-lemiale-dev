En The Zebra, promoví la creación del rol de ingeniero analítico como parte clave de los equipos de Datos e Ingeniería. Mi trabajo inicial como "analista técnico", implementando dbt junto con Snowflake, sentó las bases para el equipo de ingeniería analítica.

Desde entonces, he trabajado continuamente para mejorar el entorno de datos con nuevas herramientas y marcos como Fivetran para ETL prediseñado, funnel.io para la gestión de integraciones API, y Prefect para la orquestación de trabajos de actualización de datos.

Mi mayor orgullo dentro de esta organización de datos fue colaborar con el equipo de ingeniería de datos para diseñar un framework de pruebas unitarias SQL y un framework de pruebas de regresión, aumentando la confianza en los despliegues y garantizando la integridad de los datos.
